# 🚀 Task 1: HTML Structure & Server Interaction

## 📌 Objective

To understand basic HTML structure and implement server-side interaction using Node.js and Express.

## 🛠️ Features

* Created a user input form using HTML
* Built a basic Node.js server using Express
* Handled form submission on the server
* Implemented server-side rendering (EJS)

## ⚙️ Technologies Used

* HTML
* Node.js
* Express.js
* EJS

## ▶️ How to Run

# Initialize the project
npm init -y

# Install the essentials
npm install express ejs

# Start the server
node app.js
